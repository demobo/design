Introduction

Sunrise is a responsive coming soon page with a fullscreen background image.

Requirements

This theme has been tested in the following browsers: Chrome 23, Firefox 17, Safari 5, Internet Explorer 8 and 9, Android 2.3 Browser, iPad Safari, Opera Mobile and Chrome for Mobile.

Subscription form requires a hosting with PHP 5.2+ and a Javascript to be turned on in user's browser.

Package Contents

Zip package contains 3 folders:

docs
Contains current documentation and its assets.
psd
Contains PSD background template.
www
Contains all files to be uploaded to your web server.

Getting Started

At first, you need to customize your page. This process is described in the next section. Then, upload all files from www folder to your server and you are done.

How Subscription Form Works

Any visitor of your page is able to submit his email to subscribe for news. The subscription form uses javascript to send AJAX request to subscribe.php script. The script validates submitted email, and if it is valid, the script saves the address to subscription_list.txt file and sends an email (which is set up in email.config.php) to the user.

Customization

All customization is done in two configuration files: main.config.js and email.config.php. The first one allows you to change countdown date and messages which are showed to a user when he submits his email address. The second one is reponsible for subscription email settings.
